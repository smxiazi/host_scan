# host_scan

多线程host碰撞，发现更多资产。

把域名放在host.txt 把ip放在ip.txt，跑出的结果存放在ok.txt里面。

一开始输出（ok.txt）的内容可能都是脏数据，需要更关注后面输出(ok.txt)的那些数据！！！
